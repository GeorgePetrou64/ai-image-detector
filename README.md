# AI Image Detector – EfficientNet Transfer Learning

## Overview
Binary classification system that detects whether an image is AI-generated or real using EfficientNetB0 transfer learning.

## Features
-Config-driven training (config.yaml)
-two-stage transfer learning (freeze → fine-tune)
-Automatic threshold tuning (validation-based)
-Per-run experiment tracking (models/run_YYYYMMDD_HHMMSS/)
-SavedModel export for production inference
-Single-image prediction script
-Smoke test script (5 real + 5 fake samples)
-Dataset leakage detection script
-Reproducible environment via pip-tools

## Model Architecture

-Backbone: EfficientNetB0 (ImageNet pretrained)
-Head: GlobalAveragePooling → Dropout → Dense(1, sigmoid)
-Loss: Binary Crossentropy
-Optimizer: Adam
-Metrics: Accuracy, Precision, Recall, AUC

## Optional 2-Stage Training

-Train classifier head (backbone frozen)
-Unfreeze last N backbone layers and fine-tune at lower learning rate

## Dataset Structure

## Experiment Workflow

-Train baseline (frozen backbone)
-Enable fine-tuning (unfreeze last 10–30 layers)
-Tune learning rate
-Monitor validation AUC
-Select threshold from validation
-Evaluate on test set
